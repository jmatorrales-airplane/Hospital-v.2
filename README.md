# Hospital
## Ejercicio practico

```
	Ejercicio que registra un nuevo paciente, lo asigna a una cama durante el tiempo que esta ingresado
	y despues se le puede añadir nuevas pruebas, medicamentos y/o diagnosticos, o dar de alta al paciente.
```

### Funciones de la aplicación

#### Registrar al paciente
Se pide una serie de datos para registrar al paciente y se pide a que cama se le va a asignar.

#### Dar de alta al paciente
Indicando la cama que ocupa el paciente, recoge los datos del paciente y borra todo el registro.

#### Mostrar datos del paciente
Indicando la cama que ocupa el paciente, muestra todos los datos del paciente.

#### Añadir diagnostico
Indicando la cama que ocupa el paciente, permite añadir un diagnostico a la ficha del paciente.

#### Añadir medicamentos
Indicando la cama que ocupa el paciente, permite añadir un medicamento de la lista a la ficha del paciente.

#### Añadir pruebas
Indicando la cama que ocupa el paciente, permite añadir una nueba prueba a la ficha del paciente.

#### Mostrar camas
Muestra todas las camas ocupadas por algún paciente.

#### Salir
Finaliza el programa por completo.
